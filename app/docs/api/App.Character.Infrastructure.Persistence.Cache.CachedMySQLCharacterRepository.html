    Class App\Character\Infrastructure\Persistence\Cache\CachedMySQLCharacterRepository

    implements CharacterRepository

    This class is a wrapper around the MySQLCharacterRepository. It caches the results of the findAll, find save and methods. It uses a Redis instance to cache the results. It also uses a LoggerInterface to log the cache hits and misses.
